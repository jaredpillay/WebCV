## How to Use

1. **Update Personal Information**
   - Edit "Your Name" in the navbar
   - Update the hero section title and subtitle
   - Replace placeholder text with your actual content

2. **Customize Skills**
   - Modify the 4 skill categories in the Skills section
   - Change icons (uses Font Awesome)
   - Add/remove skills as needed

3. **Add Work Experience**
   - Update job titles, companies, dates
   - Add descriptions of your responsibilities and achievements

4. **Add Education**
   - Include degrees and certifications
   - Add institutions and graduation years

5. **Update Contact Information**
   - Replace email address
   - Update LinkedIn profile URL
   - Add GitHub or other profiles

6. **Customize Colors**
   - Edit CSS variables in `styles.css` (lines 8-15)
   - Primary color: `--primary-color`
   - Secondary color: `--secondary-color`
   - Text colors and backgrounds

## Deployment

### Deploy to Netlify (like the reference)
1. Create a GitHub repository
2. Push your files to GitHub
3. Go to netlify.com
4. Click "New site from Git"
5. Connect your GitHub repo
6. Deploy!

### Deploy to Vercel
1. Create a GitHub repository
2. Go to vercel.com
3. Create new project from Git
4. Select your repository
5. Deploy!

### Deploy Locally
Just open `index.html` in your browser - no build process needed!

## Customization Tips

### Change Primary Color
In `styles.css`, find the `:root` section:
```css
:root {
    --primary-color: #2563eb;  /* Change this blue color */
    --secondary-color: #1e40af;
    /* ... other colors ... */
}
```

### Add More Sections
Simply copy an existing section structure and modify:
1. Add a section with an ID
2. Add corresponding navigation link
3. Style using existing classes

### Make It Your Own
- Change fonts in body styles
- Adjust spacing and padding
- Modify hover effects
- Add your own images
- Customize the animations
